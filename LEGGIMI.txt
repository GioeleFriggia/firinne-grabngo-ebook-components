INSTALLAZIONE
1. Estrai TUTTO il contenuto dello ZIP direttamente dentro:
   C:\Users\gioel\Desktop\firinne-grabngo-ebook-components

2. Apri quella cartella con VS Code.

3. Apri il terminale PowerShell di VS Code.

4. Esegui:
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

5. Esegui:
   .\apply-update.ps1

6. Avvia il progetto:
   npm run dev

7. Nel browser esegui Ctrl + F5.

8. Controlla una scheda prodotto e la funzione Print / PDF.
