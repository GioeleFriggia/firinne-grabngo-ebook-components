FIRINNE - AGGIORNAMENTO FOTO PRODOTTI

Questo pacchetto contiene 7 immagini e uno script PowerShell che:
1. copia le immagini in public/images;
2. collega sei nuove immagini alle rispettive schede prodotto;
3. sostituisce asian-noodle-real.png mantenendo il percorso già usato dal progetto.

ISTRUZIONI
1. Estrai questa cartella.
2. Apri PowerShell nella cartella principale della repository:
   C:\Users\gioel\Desktop\firinne-grabngo-ebook-components
3. Esegui:
   PowerShell -ExecutionPolicy Bypass -File "PERCORSO_COMPLETO\firinne-product-photos-update\install-product-photos.ps1"
4. Avvia il progetto con:
   npm run dev
5. Controlla le sette schede.
6. Esegui i comandi Git mostrati dallo script.
